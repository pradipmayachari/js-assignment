The Differences Between var,let and const

var:- var variables can be updated and re-declared within its scope
var variables are initialized with undefined
example.>console.log(x); var x=5; console.log(x);

let:-let variables can be updated but not re-declared
let variables are not initialized.
example.console.log(x); let x=5; console.log(x);

const:-const variables can neither be updated nor re-declared.
They are all hoisted to the top of their scope.
const variables are not initialized.
example. const x = 12; x = 13; x += 1;
